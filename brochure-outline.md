# 2단 브로셔 아웃라인

## 브로셔 기본 정보

- **크기**: A4 (210 x 297mm) 또는 Letter (8.5" x 11")
- **접힘 방식**: 반으로 접은 2단 형태
- **페이지 구성**: 앞면/뒷면 총 4개 패널

## 패널 구성

### 📄 **패널 1 (외부 앞면/커버)**

- **목적**: 첫인상, 브랜드 인지
- **구성 요소**:
  - 로고/브랜드명
  - 메인 헤드라인
  - 키 비주얼 이미지
  - 간단한 태그라인
- **디자인 포인트**: 시선을 끄는 강력한 비주얼

### 📄 **패널 2 (내부 왼쪽)**

- **목적**: 상세 정보 제공
- **구성 요소**:
  - 서비스/제품 설명
  - 주요 특징/장점
  - 이미지 또는 일러스트
  - 부제목과 본문
- **디자인 포인트**: 읽기 쉬운 레이아웃

### 📄 **패널 3 (내부 오른쪽)**

- **목적**: 추가 정보 및 신뢰성 구축
- **구성 요소**:
  - 고객 후기/추천사
  - 회사 소개
  - 주요 통계/수치
  - FAQ 또는 추가 서비스
- **디자인 포인트**: 신뢰감을 주는 요소 강화

### 📄 **패널 4 (외부 뒷면)**

- **목적**: 행동 유도 및 연락처
- **구성 요소**:
  - 연락처 정보
  - Call-to-Action (CTA)
  - QR 코드
  - 소셜 미디어 링크
  - 주소/지도
- **디자인 포인트**: 명확한 행동 지침

- 서브 컬러: [보조 색상]
- 텍스트: [가독성 높은 색상]
- 배경: [깔끔한 베이스 색상]

### 📝 **타이포그래피**

- **헤드라인**: 굵고 명확한 폰트 (24-32pt)
- **서브헤드**: 중간 굵기 (18-24pt)
- **본문**: 읽기 쉬운 폰트 (12-16pt)
- **캡션**: 작은 크기 (10-12pt)

### 📐 **레이아웃 원칙**

- **여백**: 충분한 화이트 스페이스
- **정렬**: 일관된 그리드 시스템
- **계층**: 명확한 정보 위계
- **균형**: 시각적 무게 분산

## 인쇄 고려사항

### 🖨️ **기술적 요구사항**

- **해상도**: 300 DPI 이상
- **컬러 모드**: CMYK
- **블리드**: 각 면 3mm 추가
- **폰트**: 아웃라인 처리 또는 임베드

### 📏 **접힘선 및 여백**

- 접힘선에서 5mm 이상 여백
- 중요한 텍스트는 가장자리에서 충분한 거리
- 양면 인쇄 시 앞뒤 정렬 확인
